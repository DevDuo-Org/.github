### 🌟 Featured project

**[Awesome Media Controller](https://github.com/DevDuo-Org/awesome-media-controller)** — a frosted-glass MPRIS media controller for the GNOME top bar: album art, animated EQ bars, six built-in themes, and a full popup card with scrubbing, volume, and multi-player switching. Works with any MPRIS2 player (Spotify, Firefox, VLC, and more).
